# Texture Bot MVP

### Возможности
- Приём фото через Telegram
- Генерация seamless tile
- PNG 2K diffuse карта
- Быстрая отправка результата

### Стек
- Node.js
- Telegraf
- Vercel Serverless
- Sharp

### Запуск
npm install
npm run bot

### .env пример
BOT_TOKEN=твой_telegram_token
VERCEL_ENDPOINT=https://your-vercel-domain/api/generate
